Frontend Template Blocks

This site hosts installable registry JSON for reusable blocks. Use the command below in any initialized shadcn project.

Install One Block

Use a direct item URL.

shadcn add https://curiousbus.github.io/frontend-template-blocks/r/simple-hero.json
